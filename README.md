# Customer Segmentation for Targeted Marketing

This project focuses on customer segmentation based on credit card usage behavior using unsupervised machine learning techniques.

## Objective
Identify homogeneous customer groups to support targeted marketing strategies.

## Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Output
- Customer clusters based on spending and repayment behavior
- Behavioral profiling of each segment
- Insights for marketing strategies
